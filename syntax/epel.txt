EPEL (Extra Packages for Enterprise Linux) is open source
and free community based repository project from Fedora
team which provides 100% high quality add-on software
packages for Linux distribution including RHEL (Red Hat
Enterprise Linux), CentOS, and Scientific Linux.  Epel
project is not a part of RHEL/Cent OS but it is designed
for major Linux distributions by providing lots of open
source packages like networking, sys admin, programming,
monitoring and so on. Most of the epel packages are
maintained by Fedora repo.
--www.tecmint.com/
